# Origami

Origami is a group of services, components, and tools used to help design & build digital products for FT brands.

## About

Origami is a collaborative project by and for the FT which includes services, components, and tools used to help design & build digital products for FT brands.

[Origami Project Mission & Vision Statements](https://origami.ft.com/documentation/principles/vision-and-mission/)

## Designers

You can browse live demos and documentation for Origami components in [the component registry](https://registry.origami.ft.com/components).

The Customer Products design team own complimentary tooling/documentation for Newsroom products.

This includes further [design system documentation](https://www.notion.so/prd-newsroom/Design-System-35c1fc00fc9d404c8a31a374e2b8c8e4) and a [figma library](https://www.figma.com/files/938480807921629744/team/938730024138282825/Design-Systems?fuid=840167457508775739).
## Developers

To jump straight in and include Origami components in your project, see our [tutorials page](https://origami.ft.com/documentation/tutorials/).

If you would like to learn about Origami components in more detail see our [component documentation](https://origami.ft.com/documentation/components/).

For an overview of all the tools and services we offer see our [documentation page](https://origami.ft.com/documentation/).
