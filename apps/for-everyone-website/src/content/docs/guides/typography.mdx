## Typographic Scale

We use a single typographic scale for all fonts. Our scale ranges from -2 to 12. Consider scale 0 the baseline, with ascending and descending values in each direction. Avoid using styles below zero for blocks of text premium user experience.

* Our legacy typographic scale uses the same values as our latest scale. See our legacy Foundations library and legacy o-typography component in Storybook.

### Metric 2 (Variable Font, VF)

Modern Origami introduces the Metric2 font family as a variable font (a incremental font weights). We use a single typographic scale for all fonts.

Metric2 replaces legacy Metric font files. Metric2 renders larger than Metric for the same font size, therefore you may need to use a smaller scale to align with legacy Metric typography.

## Font Weight

Font weight can be either be used to express hierarchy or urgency of information.

For instance:
* In editorial, Lighter font weights. e.g regular is used to depict a post that is more of a feature and less of a main news while semi-bold and bold are used for a post like live news that is a main news.

* In UI design, all heading levels use semi-bold weights while regular is usually used for sub-titles and body copy.
