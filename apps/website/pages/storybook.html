Interactive component demos and documentation.

We are aiming to replace old origami registry that is currently used to view demos and documentation for Origami components with new storybook interface. The reasoning behind choosing Storybook is that it is a well known and widely used tool for documenting and making highly interactive UI demos.

Storybook is still work in progress project and not all components have demos. If you don't find the component in storybook, for the moment you will need to reference our registry instead.

We wrote more about this in our October 2021 newsletter.
