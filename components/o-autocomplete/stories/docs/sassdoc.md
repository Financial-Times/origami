# o-autocomplete Sass Documentation
## Mixins
### oAutocomplete
Output all oAutocomplete features


| Parameter | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| opts | Map | () |A map of options to configure the output |
#### Examples
##### Example 1
```scss
@include oAutocomplete($opts: (
	// your opts here
))
```
