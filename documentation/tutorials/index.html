    Origami Tutorials#

    This section of the site offers step-by-step tutorials for people who are learning to use Origami. We also run through some of these in the Origami Workshop, if you’re new to the FT or have been here a while but could use a recap, then please contact the team.

    Include Components Using the Origami Build Service#

    A step-by-step tutorial which teaches you how to use Origami components via the Origami Build Service. Learn how to build web pages using the Origami Build Service

    Create A New Origami Component#

    A step-by-step tutorial which teaches you how to build and deploy a new Origami component. Learn how to create an Origami component

    Include Components Using a Package Manager (npm)#

    A step-by-step tutorial which teaches you how to use Origami components via a package manager, compiling them on your local machine. Learn how to build web pages with locally-installed Origami components
