Origami Storybook

Interactive component demos and documentation.

We are aiming to replace the Origami Registry, that is used to view Origami component demos and documentation, with a new Storybook interface. The reasoning behind choosing Storybook is that it is a well known and widely used tool for documenting and making highly interactive component demos.

Storybook is still work in progress project and not all components have demos. If you don't find the component in storybook, for the moment you will need to reference the Origami Registry instead.

We wrote more about this in our October 2021 newsletter.
